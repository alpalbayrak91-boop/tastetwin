# TasteTwin Chrome Extension

1. Chrome'da `chrome://extensions` ac.
2. Sag ustteki `Gelistirici modu` anahtarini ac.
3. `Paketlenmemis oge yukle` / `Load unpacked` secip bu klasoru sec:
   `C:\Users\morte\OneDrive\Belgeler\lb\extension`
4. Kendi Letterboxd profilini ac, eklenti ikonuna tikla ve taramayi baslat.
5. TasteTwin uygulamasinda ayni kullanici adi ile `Hesabi ve takipleri cek` butonuna bas.

Eklenti sadece acik Letterboxd tarayici oturumundan gorunur takipci ve takip edilen listelerini okur. Sifre istemez veya kaydetmez.
